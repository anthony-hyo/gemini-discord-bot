A lightweight **Java** Discord bot framework using **JDA** (Java Discord API).  
Features include command handling, slash commands, button interactions, AI integration via Gemini API, database persistence, and conversation memory.

---

## Features

- Slash command and prefix command support.
- Button interaction support.
- **AI-powered responses using Gemini API**.
- Conversation memory per channel.
- Database support (MariaDB + ActiveJDBC + HikariCP + Ehcache).
- Event-driven architecture for easy extension.
